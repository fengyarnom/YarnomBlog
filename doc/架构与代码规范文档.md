### 代码规范

#### python代码命名规范

##### 变量

应当采用全部小写，如若有多个单词，则用 `_`（下划线）隔开

```
a = 10
my_name = "Yarnom"
```



##### 函数

应当采用小写命名，如若有多个单词，则第一个单词小写，后面单词首字母大写

```python
def function():
    pass
def functionName():
    pass
```
